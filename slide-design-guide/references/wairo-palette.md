# 和色パレット集（NIPPON COLORS / 和色大辞典 由来）

ランダムなhexではなく**名前のある日本の伝統色**から取ると、低彩度で品のある
「日本のビジネス資料らしいトーン」になる。各セットは **ベース70 : メイン25 : アクセント5** で使う。
ベースは純白 `#FFFFFF` より **生成り `#FBFAF5`** にすると温度が出る（和・高級で特に有効）。

## 使い方
1. SlideLandで決めたテイスト×色から、近いセットを1つ選ぶ。
2. ベース／メイン／アクセントの役割を固定（色を増やさない＝無彩色＋3色以内）。
3. アクセントは画面の5%以下、フォーカスしたい1要素にだけ。
4. 文字コントラストは必ず WCAG AA（4.5:1）を確認（濃色面の上は白/生成り、淡色面の上は墨）。

## セット一覧（テイスト別）

### 信頼感 / 決算・IR（藍系）
- ベース: 生成り `#FBFAF5` ／ 白練 `#EAEAEA`
- メイン: 藍 `#165E83` ／ 留紺 `#1F2D54`
- アクセント: 縹（はなだ）`#2792C3`
- サブ無彩: 藍鼠 `#5C6266` ／ 墨 `#1C1C1C`

### 高級感 / 和・会社紹介（藍墨＋金茶）
- ベース: 生成り `#FBFAF5` ／ 象牙 `#F8F4E6`
- メイン: 藍 `#165E83` ／ 墨 `#1C1C1C`
- アクセント: 金茶 `#B89B5E`（細罫・シール・見出しの一語に）
- 差し（ごく一点）: 朱 `#C1432B`
- 見出しは明朝（japanese-typography の `--jp-mincho`）

### 爽やか / 環境・サービス（緑系）
- ベース: 白 `#FFFFFF` ／ 白緑 `#F1F7F2`
- メイン: 常磐緑 `#1E7A4D` ／ 若竹 `#2FA56B`
- アクセント: 浅葱 `#00A3AF`（1指標フォーカス）
- サブ無彩: 利休鼠 `#888E7E`

### ポップ / 事業計画（黄・暖色）
- ベース: 生成り `#FFFDF5` ／ 鳥の子 `#FFF6DE`
- メイン: 山吹 `#F8B500`（彩度を一段抑えて使う）
- アクセント: 珊瑚 `#F2774E` ／ 若草 `#4C9A5A`
- 文字: 墨 `#2B2620`（黄面の上は必ず濃色でAA確保）

### 近未来 / ダーク（紺＋ネオン）
- ベース(地): 鉄紺 `#0A0E16` ／ パネル `#121826`
- メイン: 群青 `#4C6CB3` ／ 青藤 `#7C6BF0`
- アクセント(発光): 青緑 `#34E1D6`（5%、1要素のみ発光）
- 文字: 白 `#E6ECF5` ／ ミュート `#8995A8`

### やさしい / ウェルネス（淡い暖色）
- ベース: 生成り `#FDF6EE` ／ 白 `#FFFFFF`
- メイン: 洗柿（あらいがき）`#D08C6A` ／ 鳶（とび）寄り `#C17A5A`
- アクセント: 千歳緑寄りのセージ `#8FA98C`
- 文字: 焦茶 `#3E342C`

### ダイナミック / スポーツ（赤系）
- ベース: 生成り `#F5F3F0` ／ 黒鳶 `#14110F`
- メイン: 緋（ひ）`#E11D3A`（力強い赤、面では使いすぎない）
- サブ無彩: 鈍色（にびいろ）`#8A8580`
- 文字: 墨 `#1A1A1A`（赤面の上は白）

### 端正 / 統合報告書（紺グレー＋金）
- ベース: 白 `#FFFFFF` ／ 銀鼠 `#EEF1F4`
- メイン: 藍鼠 `#2B3A4A` ／ `#3F5366`
- アクセント: 黄金 `#C9A24B`（細線・数字1点）
- 罫線: `#D5DBE1` ／ 文字: `#1B232B`

### ミニマル / モノクロ
- ベース: 白 `#FFFFFF`
- メイン: 墨 `#0A0A0A`
- サブ: 鼠 `#8A8A8A` ／ ヘアライン 白練 `#E6E6E6`
- 原則アクセント無し（強調も墨の濃淡で）
